# 页面

## 概述
页面。

## 属性

#### info

**说明**:提示信息

**类型**:String

**默认值**:-

#### title

**说明**:标题

**类型**:String

**默认值**:标题

#### icon

**说明**:图标。详情请看`cd-icon`的`type`属性。

**类型**:String

**默认值**:标题

**TODO**:支持本地图标组件。

#### breadcrumb

**说明**:是否展示面包屑导航。

**类型**:Boolean

**默认值**:true

#### contClass

**说明**:页面内容容器元素的class。

**类型**:String

**默认值**:page-cont

#### autoClosePageLoading

**说明**:是否自动关闭页面加载。

**类型**:Boolean

**默认值**:true


## 事件

#### on-change

**说明**: 	在固定状态发生改变时触发

**参数**: true | false
